Inspired

/design:clone

Create a new UI inspired by the reference design. Not pixel-perfect, but captures the essence.

Pixel Perfect

/design:clone-px

Exact 1:1 replica of the original. Extracts real assets and matches layout precisely.

/design:clone-px https://example.com

  • Real asset extraction
  • AI structure analysis
  • Section-based analysis
  • Menu verification
  • Layout verification
Multi-Page

/design:clone-site

Clone entire website with multiple pages. Discovers and clones all linked pages from the URL.

/design:clone-site https://example.com

  • Auto-discovers all pages
  • Shared CSS across pages
  • Consistent navigation
  • Preserves site structure
  • Internal links maintained

Get Started in 30 Seconds

Two commands and you're ready to go

1

Install via npm

npm install -g design-clone
2

Initialize the skill

design-clone init

Start cloning!

# In Claude Code
/design:clone https://your-favorite-site.com
+

Optional: Enable AI Analysis

Add Gemini API key to enable AI-powered structure analysis

# Add to ~/.claude/.env
GEMINI_API_KEY=your-api-key
Get your free API key

Requirements

Works on any modern system

Node.js 18+
Python 3.9+ (optional)
